# Trae Image Marker 需求文档

## 1. 项目概述

### 1.1 项目背景

Trae Image Marker 是一个基于 Electron 的跨平台桌面应用程序，专为建模师设计的图片标注和比例测量工具。建模师在进行 3D 建模时，经常需要参考真实照片或设计图，通过测量参考图中的比例关系来确保模型的准确性和真实感。该应用为建模师提供专业的图片标注和测量工具，支持多种标注类型，并能保存标注结果到标记文件中，帮助建模师快速、准确地获取参考图的比例信息。

### 1.2 项目目标

- 为建模师提供直观、高效的图片标注和比例测量界面
- 支持多种标注类型，满足建模师测量参考图比例关系的需求
- 实现标注数据的持久化存储，方便建模师保存和复用测量数据
- 支持跨平台运行（Windows、macOS、Linux）
- 提供良好的用户体验和操作便捷性，提高建模师的工作效率

### 1.3 目标用户

**主要用户群体：建模师**

建模师是本应用的核心用户群体，包括但不限于：

- **角色建模师**：需要测量人物照片中的身体比例关系（如头身比、四肢比例等）
- **场景建模师**：需要测量建筑照片或设计图中的尺寸比例和角度关系
- **道具建模师**：需要测量物体照片中的尺寸、角度和比例关系
- **动画建模师**：需要测量参考图中的动作角度和比例关系

**用户需求特点：**

- 需要快速、准确地测量参考图中的比例关系
- 需要保存测量数据以便后续参考和对比
- 需要同时处理多张参考图
- 需要直观的可视化标注来辅助建模工作
- 需要高效的工作流程以节省时间

## 2. 术语定义

### 2.1 图片

- **定义**：常见静态图片格式
- **支持的格式**：JPG、PNG、BMP、WebP等
- **不支持**：动态图片格式（如GIF等）
- **用途**：作为标注的载体，建模师在图片上添加各种标注来测量比例关系

### 2.2 标注

标注是指在图片上添加的可视化标记，用于表示特定的测量信息。本系统支持以下标注类型：

#### 2.2.1 水平线段

- **定义**：在图片上绘制的水平方向线段
- **功能**：用于标记图片中的水平位置，通过多个水平线段之间的垂直距离来测量垂直比例关系
- **建模师应用场景**：
  - 标记人物照片中的关键水平位置（头顶、下巴、肩部、腰部、膝盖、脚底等）
  - 通过多个水平线段测量头身比、身体各部位比例等垂直比例关系
  - 标记建筑照片中的楼层位置、门窗位置等，用于测量垂直比例
  - 标记物体照片中的关键水平位置，用于确定垂直比例关系
- **特点**：
  - 线段方向固定为水平
  - 支持拖拽调整端点位置
  - 支持整体拖拽移动线段

#### 2.2.2 垂直线段

- **定义**：在图片上绘制的垂直方向线段
- **功能**：用于标记图片中的垂直位置，通过多个垂直线段之间的水平距离来测量水平比例关系
- **建模师应用场景**：
  - 标记人物照片中的关键垂直位置（身体中心线、肢体边缘等）
  - 通过多个垂直线段测量肩宽、腰宽、四肢宽度等水平比例关系
  - 标记建筑照片中的柱子位置、墙体位置等，用于测量水平比例
  - 标记物体照片中的关键垂直位置，用于确定水平比例关系
- **特点**：
  - 线段方向固定为垂直
  - 支持拖拽调整端点位置
  - 支持整体拖拽移动线段

#### 2.2.3 普通量角器

- **定义**：用于测量任意角度的工具
- **功能**：测量两条线段之间的夹角
- **建模师应用场景**：
  - 测量人物照片中的关节角度（如肘关节、膝关节等）
  - 测量建筑照片中的屋顶角度、楼梯角度等
  - 测量物体照片中的各种角度关系，用于确定模型的姿态和结构
- **特点**：
  - 顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整角度顶点和边
  - 显示角度弧线

#### 2.2.4 水平量角器

- **定义**：用于测量相对于水平线的角度
- **功能**：测量线段与水平线的夹角
- **建模师应用场景**：
  - 测量人物照片中的身体倾斜角度
  - 测量建筑照片中的墙面倾斜角度
  - 测量物体照片中的倾斜角度，用于确定模型的姿态和透视关系
- **特点**：
  - 以水平线为基准，顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整线段位置
  - 显示角度弧线

#### 2.2.5 垂直量角器

- **定义**：用于测量相对于垂直线的角度
- **功能**：测量线段与垂直线的夹角
- **建模师应用场景**：
  - 测量人物照片中的身体垂直偏差角度
  - 测量建筑照片中的垂直结构倾斜角度
  - 测量物体照片中的垂直角度关系，用于确定模型的姿态和透视关系
- **特点**：
  - 以垂直线为基准，顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整线段位置
  - 显示角度弧线

### 2.3 标记文件

- **定义**：保存多张图片以及每张图片上所有标注相关信息的文件
- **关系**：一个标记文件可以包含多张图片的标注信息，每张图片对应一组独立的标注数据
- **展示方式**：在应用中，每个标签页只展示一张图片及其标注信息，多张图片通过多个标签页进行展示和切换
- **功能**：持久化存储标注数据，支持后续编辑和查看
- **内容包含**：
  - 多张图片的原始图片数据（或图片路径引用）
  - 每张图片上的所有标注的几何信息（坐标、尺寸、角度等）
  - 标注类型和属性
  - 标注样式配置（颜色、线宽、字体等）
  - 元数据（创建时间、修改时间、作者等）
- **文件格式**：建议使用 JSON 格式存储标注数据，可考虑压缩以减小文件体积

## 3. 功能需求

### 3.1 文件功能

#### 3.1.1 新建标记文件

- 弹出选择框，用户选择目录路径，填写文件名后，创建一个空白的json格式文件。
- 选择框只能处理json格式文件，不支持其他格式。
- 记录文件绝对路径。

#### 3.1.2 打开标记文件

- 弹出选择框，用户选择json文件后，读取文件内容，解析标注数据，展示在应用中。
- 选择框只能处理json格式文件，不支持其他格式。
- 记录文件绝对路径。

#### 3.1.3 保存

- 反序列化当前文件内容，把内容保存在记录的文件路径中。
- 没有新建或打开文件时，保存功能不可用

#### 3.1.4 另存为

- 弹出选择框，用户选择目录路径，填写文件名后，反序列化当前文件内容，把内容保存到新的json文件中。
- 选择框只能处理json格式文件，不支持其他格式。
- 记录新文件的绝对路径。
- 没有新建或打开文件时，保存功能不可用

#### 3.1.5 添加图片

- 没有新建或打开文件时，添加图片功能不可用
- 弹出选择框，用户选择图片文件后，读取图片内容，添加到当前标记文件，创建新的标签页
- 只支持常见静态图片格式

#### 3.1.6 删除图片

- 删除当前激活的标签页的图片和标注数据
- 没有新建或打开文件时，删除图片功能不可用

#### 3.1.7 导出图片

- 没有新建或打开文件时，导出图片功能不可用
- 将当前标签页的图片和标注导出为PNG格式
- 导出时将原始图片和所有标注合并到一张PNG图片中
- 导出尺寸应该取标注和图片两者间的最大值，标注超出图片时，背景应该是透明的。

### 3.2 撤销/重做功能

- 支持撤销操作
- 支持重做操作
- 支持最近20次操作的撤销/重做

### 3.3 标注功能

#### 3.3.1 标注编辑

- 选择标注
  - 点击标注进行选择
  - 显示选中状态（高亮、控制点）
- 移动标注
  - 拖拽标注整体移动
- 调整标注
  - 拖拽控制点调整标注尺寸或角度
- 删除标注
  - 选中标注后按 Delete 键删除
  - 右键菜单删除标注

#### 3.3.2 标注创建

- 水平线段标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注
- 垂直线段标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注
- 普通量角器标注
  - 点击画布确定顶点1
  - 移动鼠标（可自由移动），实时绘制边1（顶点1为起点，鼠标为终点），然后点击确定顶点2
  - 移动鼠标（可自由移动），实时绘制边2（顶点2为起点，鼠标为终点），最后点击鼠标，确定顶点3，创建结束
- 水平量角器标注
  - 点击画布确定顶点1
  - 移动鼠标（只能水平移动），实时绘制边1（顶点1为起点，鼠标为终点），然后点击确定顶点2
  - 移动鼠标（可自由移动），实时绘制边2（顶点2为起点，鼠标为终点），最后点击鼠标，确定顶点3，创建结束
- 垂直量角器标注
  - 点击画布确定顶点1
  - 移动鼠标（只能垂直移动），实时绘制边1（顶点1为起点，鼠标为终点），然后点击确定顶点2
  - 移动鼠标（可自由移动），实时绘制边2（顶点2为起点，鼠标为终点），最后点击鼠标，确定顶点3，创建结束

#### 3.3.3 标注样式

可修改以下标注样式：
- 线条颜色设置
- 线条宽度设置
- 标注文字字体设置
- 标注文字大小设置
- 标注文字颜色设置

### 3.4 图片功能

- 一张图片一个tab，多张图片多个tab
- 图片可以旋转（-180度到180度），图片旋转时标注不跟随旋转
- 图片可以放大（最大值800%），图片缩放时保持标注的相对位置不变
- 图片可以缩小（最小值10%），图片缩放时保持标注的相对位置不变
- 适应窗口：将图像缩放至窗口大小，保持图像比例，居中
- 实际大小：将图像缩放至原始大小，居中

### 3.5 辅助线功能

- 标注距离显示
  - 选中标注后，鼠标悬停在其他同类标注上时，显示两者的距离
- 能够同时显示所有同类标注间的距离

## 4. 用户界面需求

**重要说明：本应用的用户界面设计风格严格参考 VSCode Dark+ 主题，所有颜色、字体、交互效果均与 VSCode Dark+ 主题保持一致，为用户提供熟悉、专业的视觉体验。**

### 4.1 整体布局

- 采用上中下三部分垂直布局
- 顶部：顶部菜单
- 中上部：标签页栏
- 中间：画布区域
  - 左侧：悬浮工具栏
- 底部：底部状态栏

### 4.2 顶部菜单（Header）

#### 4.2.1 文件菜单（File）

- 新建标记文件：创建一个新的标记文件
- 打开标记文件：打开已保存的标记文件
- 保存标记文件：保存当前标记文件
- 另存为：将当前标记文件保存为新文件名
- 添加图片：向当前标记文件中添加新图片（将创建新的标签页）
- 删除图片：删除当前激活的标签页的图片和标注数据
- 导出为PNG：导出当前标签页的图片和标注为PNG图片

#### 4.2.2 编辑菜单（Edit）

- 撤销（Ctrl+Z）
- 重做（Ctrl+Y）
- 清除所有标注
- 删除选中标注（Delete）

#### 4.2.3 标注菜单（Annotation）

- 水平线段
- 垂直线段
- 普通量角器
- 水平量角器
- 垂直量角器

#### 4.2.4 图片菜单（Image）

- 放大（Ctrl++）
- 缩小（Ctrl+-）
- 适应窗口（Ctrl+0）
- 实际大小（Ctrl+1）
- 顺时针旋转（Ctrl+R）
- 逆时针旋转（Ctrl+Shift+R）
- 重置旋转（Ctrl+Shift+0）

#### 4.2.5 帮助

- 开发者工具：打开开发者工具

### 4.3 标签页栏（TabsBar）

- 显示标记文件内的所有图片：一张图片一个标签
- 标签可删除：删除标签等于删除图片
- 切换标签：点击标签后切换到对应的图片，画布区域显示该图片和该图片的标注

### 4.4 画布区域（Canvas）

- 显示图像和标注
- 悬浮工具栏：包含以下工具
  1. 标注类型创建工具
    - 水平线段
    - 垂直线段
    - 普通量角器
    - 水平量角器
    - 垂直量角器
  2. 图片旋转工具
    - 旋转
    - 重置
  3. 图片缩放工具
    - 放大
    - 缩小
    - 适应窗口
    - 实际大小
  4. 辅助线工具
    - 显示/隐藏辅助线

### 4.5 底部状态栏（Footer）

- 显示状态信息：
  - 标注文件名：当前打开的标记文件名称
  - 当前图片缩放比率：100%
  - 当前图片旋转角度：0°
  - 当前图片原始尺寸：1920 x 1080
