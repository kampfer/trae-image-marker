# Trae Image Marker 需求文档

## 1. 项目概述

### 1.1 项目背景

Trae Image Marker 是一个基于 Electron 的跨平台桌面应用程序，专为建模师设计的图片标注和比例测量工具。建模师在进行 3D 建模时，经常需要参考真实照片或设计图，通过测量参考图中的比例关系来确保模型的准确性和真实感。该应用为建模师提供专业的图片标注和测量工具，支持多种标注类型，并能保存标注结果到标记文件中，帮助建模师快速、准确地获取参考图的比例信息。

### 1.2 项目目标

- 为建模师提供直观、高效的图片标注和比例测量界面
- 支持多种标注类型，满足建模师测量参考图比例关系的需求
- 实现标注数据的持久化存储，方便建模师保存和复用测量数据
- 支持跨平台运行（Windows、macOS、Linux）
- 提供良好的用户体验和操作便捷性，提高建模师的工作效率

### 1.3 目标用户

**主要用户群体：建模师**

建模师是本应用的核心用户群体，包括但不限于：

- **角色建模师**：需要测量人物照片中的身体比例关系（如头身比、四肢比例等）
- **场景建模师**：需要测量建筑照片或设计图中的尺寸比例和角度关系
- **道具建模师**：需要测量物体照片中的尺寸、角度和比例关系
- **动画建模师**：需要测量参考图中的动作角度和比例关系

**用户需求特点：**

- 需要快速、准确地测量参考图中的比例关系
- 需要保存测量数据以便后续参考和对比
- 需要同时处理多张参考图
- 需要直观的可视化标注来辅助建模工作
- 需要高效的工作流程以节省时间

## 2. 术语定义

### 2.1 图片

- **定义**：常见静态图片格式
- **支持的格式**：JPG、PNG、BMP、WebP等
- **不支持**：动态图片格式（如GIF等）
- **用途**：作为标注的载体，建模师在图片上添加各种标注来测量比例关系

### 2.2 标注

标注是指在图片上添加的可视化标记，用于表示特定的测量信息。本系统支持以下标注类型：

#### 2.2.1 水平线段

- **定义**：在图片上绘制的水平方向线段
- **功能**：用于标记图片中的水平位置，通过多个水平线段之间的垂直距离来测量垂直比例关系
- **建模师应用场景**：
  - 标记人物照片中的关键水平位置（头顶、下巴、肩部、腰部、膝盖、脚底等）
  - 通过多个水平线段测量头身比、身体各部位比例等垂直比例关系
  - 标记建筑照片中的楼层位置、门窗位置等，用于测量垂直比例
  - 标记物体照片中的关键水平位置，用于确定垂直比例关系
- **特点**：
  - 线段方向固定为水平
  - 支持拖拽调整端点位置
  - 支持整体拖拽移动线段

#### 2.2.2 垂直线段

- **定义**：在图片上绘制的垂直方向线段
- **功能**：用于标记图片中的垂直位置，通过多个垂直线段之间的水平距离来测量水平比例关系
- **建模师应用场景**：
  - 标记人物照片中的关键垂直位置（身体中心线、肢体边缘等）
  - 通过多个垂直线段测量肩宽、腰宽、四肢宽度等水平比例关系
  - 标记建筑照片中的柱子位置、墙体位置等，用于测量水平比例
  - 标记物体照片中的关键垂直位置，用于确定水平比例关系
- **特点**：
  - 线段方向固定为垂直
  - 支持拖拽调整端点位置
  - 支持整体拖拽移动线段

#### 2.2.3 普通量角器

- **定义**：用于测量任意角度的工具
- **功能**：测量两条线段之间的夹角
- **建模师应用场景**：
  - 测量人物照片中的关节角度（如肘关节、膝关节等）
  - 测量建筑照片中的屋顶角度、楼梯角度等
  - 测量物体照片中的各种角度关系，用于确定模型的姿态和结构
- **特点**：
  - 顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整角度顶点和边
  - 显示角度弧线

#### 2.2.4 水平量角器

- **定义**：用于测量相对于水平线的角度
- **功能**：测量线段与水平线的夹角
- **建模师应用场景**：
  - 测量人物照片中的身体倾斜角度
  - 测量建筑照片中的墙面倾斜角度
  - 测量物体照片中的倾斜角度，用于确定模型的姿态和透视关系
- **特点**：
  - 以水平线为基准，顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整线段位置
  - 显示角度弧线

#### 2.2.5 垂直量角器

- **定义**：用于测量相对于垂直线的角度
- **功能**：测量线段与垂直线的夹角
- **建模师应用场景**：
  - 测量人物照片中的身体垂直偏差角度
  - 测量建筑照片中的垂直结构倾斜角度
  - 测量物体照片中的垂直角度关系，用于确定模型的姿态和透视关系
- **特点**：
  - 以垂直线为基准，顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整线段位置
  - 显示角度弧线

### 2.3 标记文件

- **定义**：保存图片以及该图片上所有标注相关信息的文件
- **关系**：一个标注文件只绑定一张图片，不存在一对多的关系
- **功能**：持久化存储标注数据，支持后续编辑和查看
- **内容包含**：
  - 原始图片数据（或图片路径引用）
  - 该图片上的所有标注的几何信息（坐标、尺寸、角度等）
  - 标注类型和属性
  - 标注样式配置（颜色、线宽、字体等）
  - 元数据（创建时间、修改时间、作者等）
- **文件格式**：建议使用 JSON 格式存储标注数据，可考虑压缩以减小文件体积

## 3. 功能需求

### 3.1 图片管理

#### 3.1.1 图片打开

- 支持打开图片（详见术语定义 2.1 图片）
- 支持拖拽打开图片文件
- 支持通过菜单打开图片文件

#### 3.1.2 图片显示

- 在画布区域显示图片
- 支持图片缩放（放大、缩小、适应窗口、100%）
- 支持图片平移（拖拽移动）
- 支持图片旋转

#### 3.1.3 图片导出

- 导出图片和标注到PNG格式
- 导出时将原始图片和所有标注合并到一张PNG图片中
- 导出分辨率使用原始图片的分辨率

### 3.2 标注功能

#### 3.2.1 标注创建

- 水平线段标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注
- 垂直线段标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注
- 普通量角器标注
  - 点击画布确定角度顶点
  - 移动鼠标确定第一条边
  - 点击确定第一条边
  - 移动鼠标确定第二条边
  - 点击完成标注
- 水平量角器标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注
- 垂直量角器标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注

#### 3.2.2 标注编辑

- 选择标注
  - 点击标注进行选择
  - 显示选中状态（高亮、控制点）
- 移动标注
  - 拖拽标注整体移动
- 调整标注
  - 拖拽控制点调整标注尺寸或角度
- 删除标注
  - 选中标注后按 Delete 键删除
  - 右键菜单删除标注
- 复制标注
  - 选中标注后复制
  - 粘贴到当前位置

#### 3.2.3 标注样式

- 线条颜色设置
- 线条宽度设置
- 标注文字字体设置
- 标注文字大小设置
- 标注文字颜色设置
- 标注显示/隐藏

#### 3.2.4 标注管理

- 标注列表显示
  - 显示所有标注类型
  - 显示标注信息（类型、位置、尺寸、角度等）
- 标注搜索
  - 按类型搜索标注
  - 按位置搜索标注
- 标注分组（可选）
  - 将相关标注分组管理
  - 支持组的显示/隐藏
- 标注模板（建模师专用）
  - 保存常用标注组合为模板
  - 快速应用标注模板到新图片
  - 预设人物比例模板（如8头身、7头身等）
  - 预设建筑比例模板
- 标注对比（建模师专用）
  - 同时显示多张参考图的标注
  - 对比不同图片中的比例关系
  - 支持标注叠加显示

### 3.3 编辑功能

#### 3.3.1 撤销/重做

- 支持撤销操作
- 支持重做操作
- 支持撤销/重做历史记录

#### 3.3.2 标注操作

- 全选标注
- 清除所有标注
- 复制标注
- 粘贴标注

### 3.4 文件管理

- 保存项目

#### 3.4.3 标签页管理

### 3.5 视图功能

#### 3.5.1 缩放控制

- 放大图片
- 缩小图片
- 适应窗口
- 实际大小

## 4. 非功能需求

### 4.1 性能需求

- 图像加载时间：普通图像（<10MB）在 2 秒内加载完成
- 标注响应时间：标注操作响应时间 < 100ms
- 缩放流畅度：缩放操作保持 60fps
- 内存占用：单个项目内存占用 < 500MB

### 4.2 可用性需求

- 界面简洁直观，易于上手
- 操作流程清晰，符合用户习惯
- 提供快捷键支持
- 提供操作提示和帮助文档
- 错误提示友好明确

### 4.3 可靠性需求

- 应用稳定性：连续运行 8 小时无崩溃
- 数据安全性：标注数据自动保存，防止数据丢失
- 错误处理：完善的错误处理机制
- 兼容性：支持主流操作系统版本

### 4.4 可维护性需求

- 代码结构清晰，模块化设计
- 遵循编码规范
- 完善的代码注释
- 便于功能扩展

### 4.5 可扩展性需求

- 支持新增标注类型
- 支持插件机制（可选）
- 支持自定义标注样式
- 支持导入/导出多种格式
