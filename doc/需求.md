# Trae Image Marker 需求文档

## 1. 项目概述

### 1.1 项目背景

Trae Image Marker 是一个基于 Electron 的跨平台桌面应用程序，专为建模师设计的图片标注和比例测量工具。建模师在进行 3D 建模时，经常需要参考真实照片或设计图，通过测量参考图中的比例关系来确保模型的准确性和真实感。该应用为建模师提供专业的图片标注和测量工具，支持多种标注类型，并能保存标注结果到标记文件中，帮助建模师快速、准确地获取参考图的比例信息。

### 1.2 项目目标

- 为建模师提供直观、高效的图片标注和比例测量界面
- 支持多种标注类型，满足建模师测量参考图比例关系的需求
- 实现标注数据的持久化存储，方便建模师保存和复用测量数据
- 支持跨平台运行（Windows、macOS、Linux）
- 提供良好的用户体验和操作便捷性，提高建模师的工作效率

### 1.3 目标用户

**主要用户群体：建模师**

建模师是本应用的核心用户群体，包括但不限于：

- **角色建模师**：需要测量人物照片中的身体比例关系（如头身比、四肢比例等）
- **场景建模师**：需要测量建筑照片或设计图中的尺寸比例和角度关系
- **道具建模师**：需要测量物体照片中的尺寸、角度和比例关系
- **动画建模师**：需要测量参考图中的动作角度和比例关系

**用户需求特点：**

- 需要快速、准确地测量参考图中的比例关系
- 需要保存测量数据以便后续参考和对比
- 需要同时处理多张参考图
- 需要直观的可视化标注来辅助建模工作
- 需要高效的工作流程以节省时间

## 2. 术语定义

### 2.1 图片

- **定义**：常见静态图片格式
- **支持的格式**：JPG、PNG、BMP、WebP等
- **不支持**：动态图片格式（如GIF等）
- **用途**：作为标注的载体，建模师在图片上添加各种标注来测量比例关系

### 2.2 标注

标注是指在图片上添加的可视化标记，用于表示特定的测量信息。本系统支持以下标注类型：

#### 2.2.1 水平线段

- **定义**：在图片上绘制的水平方向线段
- **功能**：用于标记图片中的水平位置，通过多个水平线段之间的垂直距离来测量垂直比例关系
- **建模师应用场景**：
  - 标记人物照片中的关键水平位置（头顶、下巴、肩部、腰部、膝盖、脚底等）
  - 通过多个水平线段测量头身比、身体各部位比例等垂直比例关系
  - 标记建筑照片中的楼层位置、门窗位置等，用于测量垂直比例
  - 标记物体照片中的关键水平位置，用于确定垂直比例关系
- **特点**：
  - 线段方向固定为水平
  - 支持拖拽调整端点位置
  - 支持整体拖拽移动线段

#### 2.2.2 垂直线段

- **定义**：在图片上绘制的垂直方向线段
- **功能**：用于标记图片中的垂直位置，通过多个垂直线段之间的水平距离来测量水平比例关系
- **建模师应用场景**：
  - 标记人物照片中的关键垂直位置（身体中心线、肢体边缘等）
  - 通过多个垂直线段测量肩宽、腰宽、四肢宽度等水平比例关系
  - 标记建筑照片中的柱子位置、墙体位置等，用于测量水平比例
  - 标记物体照片中的关键垂直位置，用于确定水平比例关系
- **特点**：
  - 线段方向固定为垂直
  - 支持拖拽调整端点位置
  - 支持整体拖拽移动线段

#### 2.2.3 普通量角器

- **定义**：用于测量任意角度的工具
- **功能**：测量两条线段之间的夹角
- **建模师应用场景**：
  - 测量人物照片中的关节角度（如肘关节、膝关节等）
  - 测量建筑照片中的屋顶角度、楼梯角度等
  - 测量物体照片中的各种角度关系，用于确定模型的姿态和结构
- **特点**：
  - 顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整角度顶点和边
  - 显示角度弧线

#### 2.2.4 水平量角器

- **定义**：用于测量相对于水平线的角度
- **功能**：测量线段与水平线的夹角
- **建模师应用场景**：
  - 测量人物照片中的身体倾斜角度
  - 测量建筑照片中的墙面倾斜角度
  - 测量物体照片中的倾斜角度，用于确定模型的姿态和透视关系
- **特点**：
  - 以水平线为基准，顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整线段位置
  - 显示角度弧线

#### 2.2.5 垂直量角器

- **定义**：用于测量相对于垂直线的角度
- **功能**：测量线段与垂直线的夹角
- **建模师应用场景**：
  - 测量人物照片中的身体垂直偏差角度
  - 测量建筑照片中的垂直结构倾斜角度
  - 测量物体照片中的垂直角度关系，用于确定模型的姿态和透视关系
- **特点**：
  - 以垂直线为基准，顺时针方向测量角度
  - 角度范围：0°-360°
  - 显示角度数值
  - 支持拖拽调整线段位置
  - 显示角度弧线

### 2.3 标记文件

- **定义**：保存多张图片以及每张图片上所有标注相关信息的文件
- **关系**：一个标记文件可以包含多张图片的标注信息，每张图片对应一组独立的标注数据
- **展示方式**：在应用中，每个标签页只展示一张图片及其标注信息，多张图片通过多个标签页进行展示和切换
- **功能**：持久化存储标注数据，支持后续编辑和查看
- **内容包含**：
  - 多张图片的原始图片数据（或图片路径引用）
  - 每张图片上的所有标注的几何信息（坐标、尺寸、角度等）
  - 标注类型和属性
  - 标注样式配置（颜色、线宽、字体等）
  - 元数据（创建时间、修改时间、作者等）
- **文件格式**：建议使用 JSON 格式存储标注数据，可考虑压缩以减小文件体积

## 3. 功能需求

### 3.1 图片管理

#### 3.1.1 图片打开

- 支持打开图片（详见术语定义 2.1 图片），打开的图片将添加到当前标记文件并创建新的标签页
- 支持拖拽打开图片文件
- 支持通过菜单打开图片文件

#### 3.1.2 图片显示

- 在画布区域显示图片
- 支持图片缩放（放大、缩小、适应窗口、100%）
- 支持图片旋转

#### 3.1.3 图片导出

- 导出图片和标注到PNG格式
- 导出时将原始图片和所有标注合并到一张PNG图片中
- 导出分辨率使用原始图片的分辨率

### 3.2 标注功能

#### 3.2.1 标注创建

- 水平线段标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注
- 垂直线段标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注
- 普通量角器标注
  - 点击画布确定角度顶点
  - 移动鼠标确定第一条边
  - 点击确定第一条边
  - 移动鼠标确定第二条边
  - 点击完成标注
- 水平量角器标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注
- 垂直量角器标注
  - 点击画布确定起点
  - 移动鼠标确定终点
  - 再次点击完成标注

#### 3.2.2 标注编辑

- 选择标注
  - 点击标注进行选择
  - 显示选中状态（高亮、控制点）
- 移动标注
  - 拖拽标注整体移动
- 调整标注
  - 拖拽控制点调整标注尺寸或角度
- 删除标注
  - 选中标注后按 Delete 键删除
  - 右键菜单删除标注

#### 3.2.3 标注样式

- 线条颜色设置
- 线条宽度设置
- 标注文字字体设置
- 标注文字大小设置
- 标注文字颜色设置

### 3.3 编辑功能

#### 3.3.1 撤销/重做

- 支持撤销操作
- 支持重做操作

#### 3.3.2 标注操作

- 全选标注
- 清除所有标注

### 3.4 文件管理

#### 3.4.1 标记文件

- 新建标记文件
- 添加图片：向当前标记文件中添加新图片（将创建新的标签页）
- 保存标记文件
- 打开标记文件

#### 3.4.2 标签页管理

- 多标签页支持：一个标记文件中的多张图片通过多个标签页展示
- 标签页切换：在不同图片标签页之间切换查看和编辑
- 标签页删除：删除标记文件中对应图片及其相关的标注
- 标签页命名：标签页名称默认使用图片文件名

### 3.5 视图功能

#### 3.5.1 缩放控制

- 放大图片
- 缩小图片
- 适应窗口
- 实际大小

#### 3.5.2 辅助功能

- 标注距离显示
  - 选中标注后，鼠标悬停在其他同类标注上时，显示两者的距离
  - 显示所有同类标注间的距离

### 3.6 工具栏功能

- 主页按钮：返回主页或初始状态
- 工具按钮：打开工具面板
- 设置按钮：打开设置对话框
- 保存按钮：保存当前工作
- 撤销按钮：撤销上一步操作
- 重做按钮：重做已撤销的操作
- 放大按钮：放大图像
- 缩小按钮：缩小图像

## 4. 非功能需求

### 4.1 性能需求

- 图像加载时间：普通图像（<10MB）在 2 秒内加载完成
- 标注响应时间：标注操作响应时间 < 100ms
- 缩放流畅度：缩放操作保持 60fps
- 内存占用：单个项目内存占用 < 500MB

### 4.2 可用性需求

- 界面简洁直观，易于上手
- 操作流程清晰，符合用户习惯
- 提供快捷键支持
- 提供操作提示和帮助文档
- 错误提示友好明确

### 4.3 可靠性需求

- 应用稳定性：连续运行 8 小时无崩溃
- 数据安全性：标注数据自动保存，防止数据丢失
- 错误处理：完善的错误处理机制
- 兼容性：支持主流操作系统版本

### 4.4 可维护性需求

- 代码结构清晰，模块化设计
- 遵循编码规范
- 完善的代码注释
- 便于功能扩展

### 4.5 可扩展性需求

- 支持新增标注类型
- 支持插件机制（可选）
- 支持自定义标注样式
- 支持导入/导出多种格式

## 5. 用户界面需求

**重要说明：本应用的用户界面设计风格严格参考 VSCode Dark+ 主题，所有颜色、字体、交互效果均与 VSCode Dark+ 主题保持一致，为用户提供熟悉、专业的视觉体验。**

### 5.1 整体布局

- 采用上中下三部分垂直布局
- 顶部：Header（顶部菜单）
- 中上部：TabsBar（标签页栏）
- 中间：Canvas（画布区域）
- 底部：Footer（底部状态栏）

### 5.2 顶部菜单（Header）

#### 5.2.1 文件菜单（File）

- 新建标记文件：创建一个新的标记文件
- 打开标记文件：打开已保存的标记文件
- 保存标记文件：保存当前标记文件
- 另存为：将当前标记文件保存为新文件名
- 添加图片：向当前标记文件中添加新图片（将创建新的标签页）
- 删除图片：删除当前激活的标签页的图片和标注数据
- 导出为PNG：导出当前标签页的图片和标注为PNG图片

#### 5.2.2 编辑菜单（Edit）

- 撤销（Ctrl+Z）
- 重做（Ctrl+Y）
- 清除所有标注
- 删除选中标注（Delete）

#### 5.2.3 标注菜单（Annotation）

- 水平线段
- 垂直线段
- 普通量角器
- 水平量角器
- 垂直量角器

#### 5.2.4 视图菜单（View）

- 放大（Ctrl++）
- 缩小（Ctrl+-）
- 适应窗口（Ctrl+0）
- 实际大小（Ctrl+1）

##### 5.2.5 帮助

- 开发者工具：打开开发者工具

### 5.3 标签页栏（TabsBar）

- 显示打开的图像标签页
- 支持标签页切换和删除

### 5.4 画布区域（Canvas）

- 显示图像和标注
- 悬浮工具栏：8个工具按钮
  - 主页、工具、设置、保存
  - 撤销、重做、放大、缩小

### 5.5 底部状态栏（Footer）

- 显示状态信息：
  - 状态：就绪/标注中/编辑中
  - 缩放：100%
  - 画布尺寸：1920 x 1080
  - 标注数量：X 个

### 5.6 主题

- 采用 VSCode Dark+ 主题风格
