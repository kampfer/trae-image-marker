# UI 结构总结

## 整体布局

应用采用 **上中下三部分** 的垂直布局结构，使用 Ant Design 的 `Layout` 组件实现。

```
┌─────────────────────────────────────┐
│         Header（顶部菜单）          │
├─────────────────────────────────────┤
│      TabsBar（标签页栏）           │
├─────────────────────────────────────┤
│                                 │
│  ┌────┐                          │
│  │    │                          │
│  │工具│      Canvas（画布区域）    │
│  │栏  │                          │
│  │    │                          │
│  └────┘                          │
│                                 │
├─────────────────────────────────────┤
│       Footer（底部状态栏）         │
└─────────────────────────────────────┘
```

## 详细结构

### 1. Header（顶部菜单）

- **组件**：`src/components/Header/Header.tsx`
- **样式**：`src/components/Header/Header.module.css`
- **功能**：
  - 使用 Ant Design `Menu` 组件（水平模式）
  - 显示项目标签（项目1、项目2、项目3）
  - 支持项目切换
- **样式特点**：
  - 高度：50px
  - 背景：深灰色（#1f1f1f）
  - 底部边框：暗灰色（#303030）
  - 完全复用 Ant Design 菜单样式

### 2. TabsBar（标签页栏）

- **组件**：`src/components/TabsBar/TabsBar.tsx`
- **样式**：`src/components/TabsBar/TabsBar.module.css`
- **功能**：
  - 使用 Ant Design `Tabs` 组件（可编辑卡片模式）
  - 显示打开的图像标签页（图像1.jpg、图像2.png、图像3.jpg）
  - 支持标签页切换和关闭
- **样式特点**：
  - 高度：40px
  - 背景：深灰色（#1f1f1f）
  - 底部边框：暗灰色（#303030）
  - 标签页背景：深灰色（#2a2a2a）
  - 激活标签页背景：较深灰色（#3a3a3a）
  - 激活标签页边框：蓝色（#4096ff）

### 3. Canvas（中间画布区域）

- **组件**：`src/components/Canvas/Canvas.tsx`
- **样式**：`src/components/Canvas/Canvas.module.css`
- **功能**：
  - 画布区域：显示"画布区域"占位文字
  - 悬浮工具栏：8个工具按钮
- **工具栏按钮**：
  - 主页、工具、设置、保存
  - 撤销、重做、放大、缩小
- **样式特点**：
  - 画布背景：深黑色（#0a0a0a）
  - 工具栏：悬浮在画布左侧，垂直居中
  - 使用 `FloatButton.Group` 组件
  - 按钮形状：方形（square）
  - 按钮尺寸：大尺寸（large）

### 4. Footer（底部状态栏）

- **组件**：`src/components/Footer/Footer.tsx`
- **样式**：`src/components/Footer/Footer.module.css`
- **功能**：显示状态信息
- **状态项**：
  - 状态：就绪
  - 缩放：100%
  - 画布尺寸：1920 x 1080
- **样式特点**：
  - 高度：32px
  - 背景：深灰色（#1f1f1f）
  - 顶部边框：暗灰色（#303030）
  - 文字颜色：半透明白色

## 主题配置

- **主题模式**：Dark（暗色主题）
- **配置方式**：使用 `ConfigProvider` 的 `theme.darkAlgorithm`
- **整体背景**：深黑色（#141414）
